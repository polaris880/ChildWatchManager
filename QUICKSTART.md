## 🎯 操作指南

### 设置界面
```
[ - ] 工作日 [ + ]    ← → 调整
[ - ] 周末   [ + ]    ↑ ↓ 切换行
[ - ] 单次   [ + ]
[ - ] 间隔   [ + ]
[ - ] 休息   [ + ]
[开关] 开机自启动
[开关] 启用监控
[修改密码]  [保存设置]
```

### 密码输入
```
[▲ 增加]     ← 上键增加
   [5]       ← 当前数字
[▼ 减少]     ← 下键减少

[确认该位]  [清除重置]
```

## 📊 数据说明

### 每日统计
- **今日已看**：当日总观看时长
- **当前会话**：本次观看时长
- **剩余时间**：今日可用时长

### 每周统计
- **周总时长**：本周累计观看
- **日均时长**：平均每日观看
- **奖励使用**：已使用的奖励时间

## 🔧 技术改进

1. **异常处理** - 所有数据加载都有try-catch
2. **UI刷新** - onResume时自动刷新
3. **密码安全** - SHA-256加密存储
4. **数据隔离** - 每个孩子独立统计
