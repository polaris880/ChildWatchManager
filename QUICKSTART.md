## 🎮 密码输入操作

```
[ - ]  [0]  [ + ]    ← → 调整数字
         ↑
      当前数字

[确认]  [清除]       ← 确认当前位
```

### 操作流程
1. 使用 + - 按钮调整数字（0-9）
2. 点击"确认"输入当前位
3. 重复6次完成密码设置
4. 如需重置，点击"清除"
