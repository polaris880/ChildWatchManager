## 📲 安装到当贝盒子

### 重要：先卸载旧版本！

```bash
# 1. 卸载旧版本
adb uninstall com.childwatch.manager

# 2. 安装新版本
adb install app-debug.apk

# 3. 启动
adb shell am start -n com.childwatch.manager/.MainActivity
```

## 🎮 快速操作指南

### 主界面
```
[◀]  宝宝  [▶]    ← → 切换孩子

[⚙设置] [🎁奖励] [📊报告] [⏹停止]
  ← → 选择按钮，确认键执行
```

### 领取奖励
1. 点击 **🎁 奖励**
2. 选择任务（如 📖 完成作业）
3. 点击 **领取**
4. 获得 +30分钟奖励时间

### 查看报告
1. 点击 **📊 报告**
2. 查看今日/本周统计
3. 查看习惯评估

## 🎁 奖励系统使用

### 适合场景
- ✅ 孩子完成作业后
- ✅ 户外运动回来后
- ✅ 帮忙做家务后
- ✅ 阅读绘本后

### 注意事项
- 每个任务每天只能领取1次
- 奖励时间自动加入当日时长
- 次日0点自动重置

## 📊 查看观看报告

### 报告内容
- **今日统计**：时长、次数、休息
- **本周统计**：总时长、日均、奖励使用
- **习惯评估**：智能评估和建议

### 评估标准
- 👍 **良好**：≤1小时/天
- 👌 **适中**：1-2小时/天
- ⚠️ **较长**：>2小时/天

## 🎯 首次使用流程

### 1. 设置密码
1. 启动APP
2. 点击 **⚙ 设置**
3. 设置6位家长密码

### 2. 添加孩子（默认已有一个）
1. 主界面切换孩子
2. 为每个孩子配置规则

### 3. 配置时长
1. 工作日：120分钟
2. 周末：180分钟
3. 单次：30分钟
4. 间隔：30分钟

### 4. 教孩子使用
1. 告诉孩子奖励机制
2. 引导孩子领取奖励
3. 培养时间管理意识

## 🔧 故障排除

**Q: 看不到奖励按钮？**
A: 确保已安装v2.0.0版本

**Q: 孩子数据混淆？**
A: 每个孩子数据完全独立，切换孩子即可查看

**Q: 奖励领不了？**
A: 每个任务每天只能领一次

**Q: 进度条不更新？**
A: 需要开启监控服务
