# 儿童锁 v1.1.0

当贝盒子H5 Pro 儿童观看时长管理APP（遥控器优化版）

## 🎮 版本特性

### v1.1.0 更新内容
- ✅ **APP名称**：改为"儿童锁"
- ✅ **遥控器优化**：全面支持遥控器操作
- ✅ **深色主题**：适合电视观看
- ✅ **数字选择器**：无需键盘输入
- ✅ **大按钮设计**：方便遥控器选择
- ✅ **横屏显示**：适合电视屏幕

---

## 📱 功能特性

### 核心功能
- ✅ **开机自启动** - 自动启动监控
- ✅ **家长密码保护** - 退出需验证（数字选择器输入）
- ✅ **每日时长限制** - 工作日/周末分别设置
- ✅ **单次时长限制** - 防止长时间连续观看
- ✅ **间隔休息提醒** - 强制休息保护视力
- ✅ **自动锁定** - 时间到自动锁屏
- ✅ **后台监控** - 持续运行

### 遥控器操作
- **方向键**：导航选择
- **确认键**：执行操作
- **返回键**：最小化（保持后台）
- **数字选择器**：密码和数值输入

## 🎯 操作指南

### 主界面
```
┌─────────────────────────────────────┐
│           儿童锁                    │
│                                     │
│    ● 监控中                         │
│    今日已观看: 00:30:00             │
│    当前会话: 00:15:00               │
│    剩余时间: 01:30:00               │
│                                     │
│    [⚙ 设置]    [⏹ 停止监控]       │
│                                     │
│    使用遥控器方向键导航，确认键选择  │
└─────────────────────────────────────┘
```

### 设置界面
- **上下键**：选择设置项
- **左右键**：调整数值（±10或±5）
- **确认键**：点击按钮

### 密码输入
1. 使用上下键选择数字（0-9）
2. 按确认键确认当前位
3. 重复6次完成密码设置

## 📲 安装说明

### 方法1：ADB安装（推荐）
```bash
# 1. 开启当贝盒子ADB调试
# 2. 连接设备
adb connect <当贝盒子IP>:5555

# 3. 卸载旧版本（如有）
adb uninstall com.childwatch.manager

# 4. 安装新版本
adb install app-debug.apk

# 5. 启动
adb shell am start -n com.childwatch.manager/.MainActivity
```

### 方法2：U盘安装
1. 复制APK到U盘
2. 插入当贝盒子
3. 使用文件管理器安装

### 解决"解析中"图标
如果出现"解析中"图标，请：
1. 先卸载旧版本
2. 重启当贝盒子
3. 重新安装新版本

## 🔐 密码管理

### 首次使用
1. 启动APP
2. 自动进入密码设置

## ❓ 常见问题

**Q: 如何退出APP？**
A: 输入家长密码可退出，或使用返回键最小化（保持后台监控）

**Q: 忘记密码怎么办？**
A: 卸载重装APP，重新设置密码

**Q: 如何完全关闭监控？**
A: 主界面点击"停止监控"，输入密码确认

**Q: 开机不自动启动？**
A: 检查设置中的"开机自启动"是否开启

**Q: 遥控器某些键不响应？**
A: 使用方向键和确认键操作，返回键会最小化APP
