# 儿童锁 v2.0.0 - 家庭增强版

当贝盒子H5 Pro 儿童观看时长管理APP（家庭优化版）

## 🎯 版本亮点

### v2.0.0 新增功能
- ✅ **多孩子支持** - 为每个孩子单独配置
- ✅ **奖励系统** - 完成任务获得额外时间
- ✅ **观看报告** - 详细的使用统计和分析
- ✅ **进度显示** - 直观的使用进度条
- ✅ **习惯评估** - 智能评估观看习惯
- ✅ **遥控器优化** - 全面支持电视遥控器

---

## 👨‍👩‍👧‍👦 多孩子支持

### 功能特点
- **独立配置** - 每个孩子有独立的时长限制
- **快速切换** - 主界面直接切换孩子
- **个性化设置** - 不同孩子不同规则

### 使用方法
1. 主界面使用 ◀ ▶ 按钮切换孩子
2. 进入设置为当前孩子配置规则
3. 每个孩子的数据独立统计

---

## 🎁 奖励系统

### 可获得奖励
| 任务 | 奖励时间 |
|------|----------|
| 📖 完成作业 | +30分钟 |
| 🏃 户外运动 | +20分钟 |
| 🧹 做家务 | +15分钟 |
| 📚 阅读绘本 | +20分钟 |

### 规则
- 每个任务每天只能领取一次
- 奖励时间自动加入当日可用时长
- 次日自动重置

## 📊 观看报告

### 今日统计
- 总观看时长
- 观看次数
- 休息次数

### 本周统计
- 周总时长
- 日均时长
- 奖励使用情况

### 习惯评估
- 👍 良好（≤1小时）
- 👌 适中（1-2小时）
- ⚠️ 较长（>2小时）

## 🎮 遥控器操作

### 主界面
```
[◀ 宝宝 ▶]  ← → 切换孩子

[⚙ 设置] [🎁 奖励] [📊 报告] [⏹ 停止]
  ← → 切换按钮，确认键选择
```

### 设置界面
```
[ - ] 120 [ + ]  ← → 调整数值
```

### 密码输入
```
[▲]  ← 上键增加数字
 5   ← 当前数字
[▼]  ← 下键减少数字
```

## ❓ 常见问题

**Q: 如何添加新孩子？**
A: 目前需要修改配置文件，后续版本会支持界面添加

**Q: 奖励时间什么时候重置？**
A: 每日0点自动重置

**Q: 可以禁止领取奖励吗？**
A: 可以在设置中关闭奖励功能（后续版本）

**Q: 如何查看详细报告？**
A: 主界面点击"报告"按钮

**Q: 多孩子的数据会混淆吗？**
A: 不会，每个孩子的数据完全独立
